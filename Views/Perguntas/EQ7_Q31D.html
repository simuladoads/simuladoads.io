31. Permite a criação de novas classes que se reutilizam, estendem e modificam o comportamento definido em outras classes. É chamada de classe base, possuindo uma ligação direta com esses membros chamada classe derivada. Uma classe derivada pode ter apenas uma classe base direta, este conceito se refere a:

a) Encapsulamento. b) Polimorfismo. c) Interfeces. d) Herança.
