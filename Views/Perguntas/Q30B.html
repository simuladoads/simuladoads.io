30. Assinale a alternativa CORRETA quando falamos em ciclo de vida de software e as fases: obtenção dos requisitos; projeto rápido; construção de protótipo; avaliação do protótipo; refinamento do protótipo e construção do produto:

a) É reconhecido como modelo mais antigo e mais amplamente usado da engenharia de software. b) Visa entender os requisitos do usuário, obtendo assim uma melhor definição dos requisitos do sistema. Com esses requisitos especificados em mãos, o desenvolvedor estará apto a desenvolver um modelo do software que deverá ser construído. c) Proposto por Boehm em 1988 o modelo foi reconhecido como uma forma de integrar os diversos modelos existentes naquela época e possuía a finalidade de eliminar as dificuldades desses modelo explorando cada um dos seus pontos fortes. d) Permite que o desenvolvimento do projeto seja realizado paralelamente de múltiplas partes do projeto, cada uma sendo abordada de maneira diferenciada, para isso é necessário que técnicas especificas para estimar e sincronizar cronogramas sejam utilizadas. E) Alterar o gateway da estação PROFESSORES01 para o endereço de IP 175.16.30.200.
